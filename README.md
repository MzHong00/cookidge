**Cookidge**는 Cook + Firdge의 합성어로, 냉장고를 관리하고 공유할 수 있는 냉장고 서비스와 요리 레시피 SNS를 결합한 음식 관련 통합 서비스이다.

### 기능
간략하고 읽기 쉽게 나타내자면 다음과 같다.
- 사용자 => CRUD, 구글 로그인, 팔로우, 검색, 랭킹
- 레시피 => CRUD, 필터, 정렬, 좋아요, 검색, 추천, 랭킹
- 냉장고 => CRUD, 정렬, 검색, 공유
